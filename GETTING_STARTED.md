## üêõ Troubleshooting

### Windows Won't Boot

**Wait longer** - First boot takes 30-60 seconds, sometimes up to 2 minutes.

### Apps Won't Open

1. Ensure Windows has fully booted (desktop visible)
2. Wait 5-15 seconds for app to launch
3. Try clicking Start menu again
4. Increase memory in settings (if available)

### Browser in Windows Won't Load Pages

1. Check network is enabled in settings
2. Wait for network to initialize
3. Try different websites
4. Check WebSocket relay status

### Slow Performance

1. Close other browser tabs
2. Increase memory to 2048MB
3. Disable network if not needed
4. Use "slow" CPU speed for stability
